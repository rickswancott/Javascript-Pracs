Introduction: In this task you have to choose, Copy, complete and submit ALL exercise from the list provided on the school's intranet SDD web site. Each exercise should address the following criteria:

  • All questions for each exercise must be completed.
  • All exercises should be linked using properly tested hyperlinks to the index web page provided.
  • All practicals should demonstrate the application of good user interface design.
  • All practicals should demonstrate good structured program design including internal documentation such as comments in the code.
  • All exercise and the index page should be saved at the same level in one folder labelled with your full name,
  • The final project to be submitted as advised by your teacher.

The exercises vary in mark value with the harder exercises of higher value. You should aim to do the easier exercise as each exercise builds on the next one.

The exercises are designed to gradually build your programming skill to a middle level of competency in Javascript and the use of the CANVAS HTML 5 objects.

You will find the W3School website an important source of help.

You must acknowledge, with references and hyperlinks, any code copied or cut and pasted from sites other than my own ( this site).

If you are not already familiar with a professional web development tool such as Dreamweaver  or Visual Studio Code Chrome developer edition, you can use Seamonkey.

Seamonkey is an easy to use browser with a simple built in javascript editor. There are development tool add-ons for Mozilla browsers.

You must use the index page provided and add the test links to each exercise as you complete it.
